## What it does

Creatures of Sonaria is a Roblox game built around survival and creature collection, and this script is meant to make navigating and fighting in that world easier. It includes aimbot functionality plus ESP, or Extra Sensory Perception, overlays that help surface targets and nearby points of interest. Because it is packaged as an HTML document, it can be opened directly in a modern browser without a complicated setup flow.

The 2026 release focuses on improving the targeting and detection behavior so the tool feels more responsive in everyday use. Aim assist and overlay rendering have both been adjusted for the current game environment, which helps cut down on flicker and makes the interface react more cleanly. The update is aimed at staying compatible with newer Roblox changes while keeping the UI minimal and practical.
